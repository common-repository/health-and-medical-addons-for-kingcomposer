<strong><a href="https://codenpy.com/item/nilima-health-medical-wordpress-theme/">This plugin works best with Nilima Theme - 100% Free</a></strong>

Nilima Health & Medical Addons for KingComposer is king composer addons that includes 2 home pages. It is retina ready, fully responsive and compatible with all kind of mobile devices. This item is dedicated to (you can built website based on it) and will fit for Medical Clinic, Plastic Surgery Center, Dentist cosmetologist, dermatologist, photodynamic therapy, body contouring,  skin surgery, laser treatment, medical restoration & rejuvenation etc. You can save your time and money with our WordPress theme Nilima.

We built this theme on Page Builder, a very convenient drag-n-drop Page Builder for WordPress. It should provide you with the smoothest experience while creating your website with our theme.

== Installation ==

Installing this plugin as regular wordpress plugin. First you need to install King Composer Page Builder plugin then install King Composer Page Builder.

This addon plugin works like other page builder addon plugin. Go to new page then click on King Composer Page builder then click on Elements you should see Nilima Theme menu as Name "Nilima Theme" on Menu Top, click on it you will see shortcode item list.
